第二章 Shell 命令概述

目录的组织结构

目录名 内容描述
/bin 所有用户可使用的可执行文件
/sbin 系统管理员使用的执行文件
/boot Linux 内核映像文件和与引导加载有关的文件
/dev 所有设备文件，包括字符设备和块设备
/etc 系统配置文件
/lib 共享库文件，供 /bin 和 /sbin 下的文件使用
/mnt 挂载点，常用于挂在文件系统
/proc 基于内存的文件系统，用于显示内核信息
/usr/bin 用户命令工具
/usr/include C头文件
/usr/lib 函数库
/usr/src 源代码目录
/usr/sbin 系统命令工具

输入输出重定向和管道

  • 输入重定向
    • 语法为 命令 <输入文件，如 cat </etc/fstab
  • 输出重定向
    • n>输出文件，将输出内容覆盖至重定向文件
    • n>>输出文件，将输出内容追加至重定向文件
    • n 为 1，标准输出；n 为 2，标准错误输出；n 为 &，标准输出和标准错误输出
  • 无名管道，在内存中建立文件描述，使用后将被自动释放
    • 语法为 命令1 | 命令2 | … [命令n]。功能是将命令1的标准输出作为命令2的标准输入，接着将命令2的标准输出作为命令3的标准输入……
  • 命名管道，在文件系统中建立命名管道文件，通过对文件的读写实现进程通信
    • 语法为 mkfifo [选项] 文件名

第七章 Linux 信号

信号概述

  • 信号的概念：内核与进程间通信的一种方式

间隔计时器

  • Linux 系统中默认最小的时钟间隔为 10ms
  • 进程的运行时间
    • 进程的运行有三种状态，在时间片中轮流进行
      • 用户态，执行用户地址空间中的指令
      • 内核态，执行内核地址空间中的指令
      • 睡眠，执行其他进程的时间
    • 内核提供了 3 种类型的计数器：
      • 真实时间，包含所有的三种状态
      • 虚拟时间，只包含用户态
      • 实用时间，包含用户态和内核态
