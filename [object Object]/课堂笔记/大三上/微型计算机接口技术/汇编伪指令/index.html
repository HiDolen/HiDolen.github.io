汇编伪指令

发表于2022-09-08|更新于2022-09-08|课堂笔记大三上微型计算机接口技术
|阅读量:

  • RET，子程序的返回指令，栈顶数据出栈赋值给 IP 寄存器

  • PROC，ENDP，定义程序块，用 CALL 关键字调用

  • PUSHA，将 AX CX DX BX SP BP SI DI 都入栈。与之对应的是 POPA

  • TEST，类似于 AND 做与运算，但不修改目标操作数。结果各位都为 0 时，ZF=1。可用 JZ 进行判断跳转

  • SEG，取子程序的段基址。需要取子程序的偏移地址则用 OFFSET
