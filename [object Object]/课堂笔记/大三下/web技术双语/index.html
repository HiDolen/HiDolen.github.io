第2章 HTML/XHTML

html 结构：

基本的格式：

  • 段落
    • <p> </p>
  • 换行
    • <br/>
  • 标题
    • <h1> </h1> <h2> </h2> ……
  • 空行
    • <blockquote>
  • 上标下标
    • <sub> </sub> <sup> </sup>

符号实体：

符号 实体 含义
& &amp; Ampersand
< &lt; Is less than
> &gt; Is greater than
" &quot; Double quote
’ &apos; Single quote (apostrophe)
14\rm \frac{1}{4}41​ &frac14; One-quarter
° &deg; Degree
(space) &nbsp; Nonbreaking space
© &copy; Copyright
€ &euro; Euro

  • 图像

    • <img src = "comets.jpg" alt = "Picture of comets"/>

  • 超链接

    • <a href = "C210data.html"> Information on the Cessna 210 </a>

  • 拥有超链接的图像

    • 超链接语法里套着图像语法

  • 列表（list）

    • 每项 <li> </li>
    • 在 <ul> </ul> 内的列表，每项开头是圆圈（无序列表）。支持嵌套
    • 在 <ol> </ol> 内的列表，每项开头是数字（有序列表）。支持嵌套

  • 定义列表

    • 名词项 <dt> </dt>，紧接着定义项 <dd> </dd>
    • 放在 <dl> </dl> 内

表格（table）：

  • 表格
    • 在 <table> </table> 块内
    • 标题 <caption> </caption>
    • 每一行写在 <tr> </tr> 块内
    • 第一行为头，各个头写在逐个的 <th> </th> 内
    • 一行中的第一个属性写在 <th> </th> 内
    • 一行中的每个属性写在各个 <td> </td> 内
  • 表格的行列扩展
    • <th colspan = "3"> A B C </th>，表示这个行占了 3 列
    • <td rowspan = "2"> </td>，表示这个空的格子向下扩展了一格
  • 表格的分段
    • 在 <table> </table> 段内可以再嵌套一层，达到分段的效果
    • <thead> </thead> <tbody> </tbody> <tfoot> </tfoot>

表单（form），input 标签：

  • 表单，序列化数据的传输
    • <form action = ""> </form>
  • input 标签
    • <input type = "text"> </input>
    • <input type = "text" />
    • 可选的标签：size = "10"、maxlength = 5
    • 关于 type 标签：
      • type = "text"，文本框
      • type = "password"，输入的文本会被替换
      • type = "checkbox"，复选项，复选框
        • 同类的复选项通常有相同的 name 标签
        • 可以配合 checked = "checked" 使用，让选择框默认被勾选
        • 使用 value 标签，定义发送到服务器的消息
      • type = "radio"，与 checkbox 相似，但只能选一个选项
        • 同类的单选组拥有相同的 name 标签
  • 选择（select），生成一个可展开选项的单选框
    • <select name = "groceries"> </select>
    • 每个选项 <option> </option>
    • 可选 size 标签，控制显示选项的数目。默认为 1
  • 多行文本框（textarea）
    • <textarea name = "aspirations" rows = "3" cols = "40"> </textarea>
  • 按钮（action buttons），用于提交
    • <input type = "submit" value = "Submit Form" />
    • <input type = "reset" value = "Reset Form" />
    • 前者是指令，后者是按钮上的字

音频（audio）

  • 音频
    • <audio controls = "controls"> </audio>
    • 段内部 <source src = "xx.mp3">

视频（video）

  • 视频
    • <video width = "600" height = "500" autoplay = "autoplay" controls = "controls" preload = "preload"> </video>
    • 段内部 <source src = "xx.mp4">

第3章 样式表的层级

Levels of Style Sheets，样式表层级

用来格式化 HTML 文档。

插入样式表，有三种方法：

  • Inline Level，内联
    • 块中有 style 属性
  • Document Level，内部样式表
    • 使用单独的 <style> 段定义样式，放在文档头部
  • External
    • 将 CSS 文件独立出来
    • 调用时在段中添加标签 type="text/css"

Selector Forms，选择表

  • 简单选择表
  • Class Selectors
    • 然后就可以在 <p> 段中额外添加 class = "normal" 的标签
  • Id Selectors
    • 然后就可以在任意段中额外添加 id = "section4" 的标签
  • Contextual Selectors，上下文选择器
    • 应用于所有是 ol 段子段的 li 段
  • Pseudo Classes
    • 不是根据段，而是根据情况
    • 现在，鼠标放到 h2 元素上，就会应用 style
  • The Universal Selector
    • 对所有元素都启用

Font Properties，字体参数

确定字体的参数们。不同类的参数用分号隔开。

  • font-family: Arial, Helvetica, Futura
    • 选择第一个系统支持的字体
    • 如果字体名含有空格，就写在单引号内
  • font-size: 1.2em
  • font-style: italic
    • 确定斜体
  • font-weight: bold
    • 确定粗体
