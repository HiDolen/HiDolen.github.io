第一章 随机事件及其概率

1.1 随机事件

随机试验，简称试验，具有重复性（可重复进行）、多样性（结果不唯一且已知）、随机性（试验前不确定哪一个结果发生）。

  • 基本结果，随机试验中可能出现的各种结果。
  • 样本空间，SSS，随机试验 EEE 的所有可能结果构成的集合。
  • 样本点，样本空间中的元素，EEE 的每个基本结果。

随机事件，简称事件，是随机试验 EEE 的样本空间 SSS 的子集，即若干可能结果的集合。

  • 基本事件，事件只含有一个样本点
  • 必然事件，包含所有的样本点
  • 不可能事件，ϕϕϕ，不包含任何样本点

随机事件间的关系及计算，

  • 事件的和，A∪BA\cup BA∪B，⋃i=1∞Ai\bigcup^\infin_{i=1}A_i⋃i=1∞​Ai​
  • 事件的积，A∩BA\cap BA∩B，
    • 简记为 ABABAB
  • 事件的差，A−BA-BA−B。AAA 发生且 BBB 不发生
  • 互斥关系，A∩B=ϕA\cap B=ϕA∩B=ϕ
  • 互逆关系（对立关系），A∪B=SA\cup B=SA∪B=S 且 A∩B=ϕA\cap B=ϕA∩B=ϕ
    • 称 AAA 与 BBB 互为 逆事件，或两者互为 对立事件
    • 记为 A=B‾A=\overline{B}A=B 或 B=A‾B=\overline{A}B=A

运算规律，

  • 分配律，A∪(B∩C)=(A∪B)∩(A∪C)A\cup(B\cap C)=(A\cup B)\cap(A\cup C)A∪(B∩C)=(A∪B)∩(A∪C)

随机事件的概率，P(A)P(A)P(A)。

古典概率模型，简称古典模型，又称等可能概率模型，这种随机试验的基本特点是：样本点（基本事件）有限、每个基本事件发生的可能性相同。

条件概率，P(A∣B)P(A|B)P(A∣B)，在 BBB 发生的条件下 AAA 发生的概率。

样本空间 SSS 的一个划分，将 SSS 分为两两互不相容的一组事件。

  • 先验概率，直接通过统计获得的事件概率
  • 后验概率，根据某个事件的发生，计算条件概率，得到的事件概率
