正则表达式

匹配字符

  • .，匹配任意 1 个字符。若要匹配 . 本身需要 \.
  • []，匹配列举的字符，如 [a-zA-Z] 所有字母，[a-zA-Z0-9] 所有字母和数字
  • \d，匹配数字
  • \D，匹配非数字
  • \s，匹配空白（空格、TAB）
  • \S，匹配非空白
  • \w，匹配整个单词字符，包括字母、数字、下划线
  • \W，匹配非单词字符
  • ^，匹配行结尾。例如 ^abc 表示匹配每行开头的 abc
  • $，匹配行结尾。例如 abc$ 表示匹配每行结尾的 abc

其他：

  • [\u4e00-\u9fa5]，匹配汉字

匹配数量

  • +，在 + 之前的符号至少出现一次
  • *，在 * 之前的符号可以不出现，也可以出现多次
  • ?，在 ? 之前的符号最多出现一次，即出现 0 次或 1 次
  • ^，取反。例如 [^ABC]，意为匹配除了 A B C 的所有字母
  • {m}，前一个字符出现 m 次
  • {m,}，前一个字符至少出现 m 次
  • {m,n}，前一个字符出现 m 到 n 次

非捕获元。用于匹配，但不一定包含于匹配结果中。

  • ?:，例如 aa(?:bb|cc)，等于 aabb|aacc
    • 比起 aa(bb|cc)，这里括号内的内容只用来匹配，而不获取匹配内容，后续不能用分组的方式获取括号内的匹配内容
    • 若直接 aa(bb|cc)，其中 bb|cc 这个分组就会拥有组号，被缓存 。在 Python 中使用 findall() 函数时默认使用第 0 组号的内容，就会导致错误
    • 表达式中有几个 () 就会匹配几个字符串
  • ?=，正向肯定匹配。字符参与匹配，但不会被包含在结果中
    • 例如用 w(=?1|2) 去匹配字符串 w1w2w3，匹配到 w1 w2，匹配结果为 w w
  • ?!，正向否定匹配。与 ?= 相比，指定的字符串会被排除匹配
    • 例如用 w(?!1|2) 去匹配字符串 w1w2w3，匹配到 w3，匹配结果为 w
  • ?<=，反向肯定匹配。与 ?= 相比，指定的字符串不是在后面而是放在前面匹配
    • 例如用 (?<=1|2)w 去匹配字符串 w1w2w3，匹配到 1w 2w，匹配结果为 w w
  • ?<!，反向否定匹配
