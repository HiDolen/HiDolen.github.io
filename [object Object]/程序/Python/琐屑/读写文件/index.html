读写文件

基本用法：

在 r+ 模式下清空文件

seek() 可选第二个参数：

  • 0（默认），表示绝对位置
  • 1，相对当前位置
  • 2，相对于文件结尾的位置

truncate()，可选一个参数，表示要截断多少个字符。

模式参数

mode常用的模式，

  • r，只能读取
  • r+，读取文件，末尾追加内容
  • w，只能写入。会先清除原先的内容
  • w+，表示可以对文件进行读写双重操作。会先清除原先的内容
  • a，与 r+ 相同

当你需要以字节（二进制）形式读写文件时，只需要在mode参数中追加’b’即可，

  • rb，以二进制格式打开一个文件，用于只读
  • wb，以二进制格式打开一个文件，用于只写
  • ab，以二进制格式打开一个文件，用于追加
  • wb+，以二进制格式打开一个文件，用于读写
