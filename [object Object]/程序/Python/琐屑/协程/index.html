一些注意事项

  • 同时只能存在一个eventloop对象。但这不能阻止新建一个线程，为线程分配新的eventloop对象。
    • 创建eventloop：new_event_loop()
    • 设置eventloop：set_event_loop(myLoop)
    • 运行eventloop：myLoop.run_forever或myLoop.run_until_complete
    • asyncio.run_coroutine_threadsafe(coroutine，myLoop) ，为线程中的eventloop添加协程函数
      • 返回一个concurrent.futures.Future类的对象，可以在稍后用result = future.result() 获得函数运行结果
    • 关闭协程：myLoop.call_soon_threadsafe(myLoop.stop)
