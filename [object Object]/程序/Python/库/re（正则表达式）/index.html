re（正则表达式）

匹配一个符合项

re.match()

起始位置开始匹配，若不是起始位置匹配成功，则返回 None，否则返回匹配对象（re.Match）。

re.search()

匹配第一个符合项。成功则返回匹配对象，失败则返回 None

匹配所有符合项

re.findall()

参数 pos，可指定开始位置。默认为 0
参数 endpos，可指定结束位置。默认为字符串长度

返回列表。若有多个匹配模式，则返回元组列表。

re.finditer()

与 re.findall() 类似。这个函数会返回一个迭代器，用 for…in… 语句会得到 Match 对象。

re.split()

根据匹配结果，切割字符串。返回列表。

不如字符串的 split() 直观。建议用之前先试试 re.split() 结果怎样。

替换所有符合项

re.sub()

用于替换字符串中的匹配项。

正则表达式对象

re.compile()

生成一个正则表达式对象，对象包含函数 match() search() findall() 等等。
