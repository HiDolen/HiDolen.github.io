广告牌

广告牌技术（Billboarding），根据视角方向来旋转一个被纹理着色的多边形，使得多边形好像总是对着摄像机。

本质是构建旋转矩阵。根据以往的知识，旋转矩阵需要 3 个基向量，在这里通常是表面法线（normal）、指向上的方向（up）、指向右的方向（right）。
另外，额外需要一个锚点（anchor location） 来固定位置。

假设法线方向总是指向视角方向，即是已知的、固定的。首先计算指向右的方向：

right=up×normal\pmb{right}=\pmb{up}×\pmb{normal} rightright=upup×normalnormal

对向右方向归一化后，就可以同理得到向上的方向了：

up′=normal×right\pmb{up}^{'}=\pmb{normal}×\pmb{right} upup′=normalnormal×rightright

接下来开始写 Shader。

同样需要 "DisableBatching"="True" 禁止批处理，因为我们需要物体在模型空间下的位置来作为锚点。

文件见 Chapter11-Billboard.shader。

注意事项

"DisableBatching"="True" 会增加 Draw Call；

使用内置 Diffuse 等包含的阴影 Pass 无法得到正确的阴影，因为这些 Pass 没有进行顶点动画。需要提供自定义的 ShadowCaster Pass，即 LightMode 为 ShadowCaster 的 Pass。
拥有阴影的 Shader 代码见文件 Chapter11-VertexAnimationWIthShadow.shader。
