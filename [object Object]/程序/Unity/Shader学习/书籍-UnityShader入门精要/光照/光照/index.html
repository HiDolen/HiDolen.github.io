光照

发表于2022-09-08|更新于2022-09-08|程序UnityShader学习书籍-UnityShader入门精要光照
|阅读量:

一些名词

  • 辐照度（irradiance），量化光照强度。

标准光照模型

标准光照模型中，摄像机的光线分为 4 个部分：

  • 自发光（emissive）
  • 高斯反射（specular），用来描述当光线从光源照射到模型表面时，表面会在完全镜面反射方向散射多少辐射量
  • 漫反射（diffuse）
  • 环境光（ambient），用于描述其他所有的间接光照

环境光

物体可以被间接光照（indirect light） 所照亮，物体会受到其它物体漫反射光线的照射。在标准光照模型中，使用一种被称为环境光的部分来近似模拟间接光照。

环境光的计算非常简单，通常是一个全局变量，即场景中的所有物体都使用这个环境光。

自发光

光线不经过物体的反射，直接进入摄像机。标准光照模型用自发光来计算这个部分的贡献度。

漫反射

漫反射光照符合兰伯特定律（Lambert’s law） ：反射光线的强度与表面法线和光源方向之间夹角的余弦值成正比。计算如下：

cdiffuse=(clight⋅mdiffuse)max(0,n^⋅l^)\boldsymbol c_{diffuse}=(\boldsymbol c_{light}·\boldsymbol m_{diffuse})\rm max(0,\hat{\boldsymbol n}·\hat{\boldsymbol l}) cdiffuse​=(clight​⋅mdiffuse​)max(0,n^⋅l^)

其中，n^\hat{\boldsymbol{n}}n^ 是表面法线，l^\hat{\boldsymbol{l}}l^ 是指向光源的单位矢量，mdiffuse\boldsymbol m_{diffuse}mdiffuse​ 是材质漫反射颜色，clight\boldsymbol c_{light}clight​ 是光源颜色。

高光反射

高光反射是一种经验模型，也就是说，高光反射不完全符合真实的高光反射现象，只是看起来像。

用 r^\hat{\boldsymbol{r}}r^ 表示反射方向，n^\hat{\boldsymbol{n}}n^ 代表法线，l^\hat{\boldsymbol{l}}l^ 代表入射光入射方向的反向方向，有：

r^=2(n^⋅l^)n^−l^\hat{\boldsymbol{r}}=2(\hat{\boldsymbol{n}}·\hat{\boldsymbol{l}})\hat{\boldsymbol{n}}-\hat{\boldsymbol{l}} r^=2(n^⋅l^)n^−l^

Phong 模型计算高光反射部分：

cspecular=(clight⋅mspecular)max(0,v^⋅r^)mgloss\boldsymbol{c}_{specular}=(\boldsymbol{c}_{light}·\boldsymbol{m}_{specular})\rm max(\it 0, \hat{\boldsymbol{v}}·\hat{\boldsymbol{r}})^{m_{gloss}} cspecular​=(clight​⋅mspecular​)max(0,v^⋅r^)mgloss​

其中，

  • clight\boldsymbol{c}_{light}clight​ 是光源的颜色和强度，mspecular\boldsymbol{m}_{specular}mspecular​ 是材质的高光反射颜色
  • v^\hat{\boldsymbol{v}}v^ 代表向着摄像机的方向
  • mgloss\boldsymbol{m}_{gloss}mgloss​ 是材质的光泽度（gloss），也被称为反光度（shininess）。值越大，高光的 “亮点” 越小

另一种实现高光反射的方法：Blinn 模型。它没有使用反射方向，而是通过视角方向 v^\hat{\boldsymbol{v}}v^ 和光照方向 l^\hat{\boldsymbol{l}}l^ 引入一个新矢量 h^\hat{\boldsymbol{h}}h^：

h^=v^+l^∣v^+l^∣\hat{\boldsymbol{h}}=\frac{\hat{\boldsymbol{v}}+\hat{\boldsymbol{l}}}{|\hat{\boldsymbol{v}}+\hat{\boldsymbol{l}}|} h^=∣v^+l^∣v^+l^​

通过 Blinn 模型计算高光的公式如下：

cspecular=(clight⋅mspecular)max(0,n^⋅h^)mgloss\boldsymbol{c}_{specular}=(\boldsymbol{c}_{light}·\boldsymbol{m}_{specular})\rm max(\it 0, \hat{\boldsymbol{n}}·\hat{\boldsymbol{h}})^{m_{gloss}} cspecular​=(clight​⋅mspecular​)max(0,n^⋅h^)mgloss​

半兰伯特模型

与上面标准光照模型所使用的漫反射的兰伯特光照模型不同，为了避免模型在光照无法到达的区域失去所有细节表现，在兰伯特光照模型的基础上做出了修改。

广义的半兰伯特光照模型的公式如下：

cdiffuse=(clight⋅mdiffuse)(α(n^⋅l^)+β)\boldsymbol c_{diffuse}=(\boldsymbol c_{light}·\boldsymbol m_{diffuse})(α(\hat{\boldsymbol n}·\hat{\boldsymbol l})+β) cdiffuse​=(clight​⋅mdiffuse​)(α(n^⋅l^)+β)

对比下来，半兰伯特光照没有使用 max 来方式点积为赋值，而是进行了 α 倍的缩放和 β 大小的偏移。

绝大多数情况下，α 和 β 值均为 0.5。这样做后，n^⋅l^\hat{\boldsymbol n}·\hat{\boldsymbol l}n^⋅l^ 的结果范围从 [-1, 1] 映射到 [0, 1] 范围内。光照射不到的地方也能够拥有明暗变化了。

float3 WorldSpaceLightDir(float4 v) 仅可用于前向渲染中。输入一个模型空间中的顶点位置，返回世界空间中从该点到光源的光照方向，即以该点为起点。内部实现使用了 UnityWorldSpaceLightDir 函数。没有被归一化
float3 UnityWorldSpaceLightDir(float4 v) 仅可用于前向渲染中。输入一个世界空间中的顶点位置，返回世界空间中从该点到光源的光照方向。没有被归一化
float3 ObjSpaceLightDir(float4 v) 仅可用于前向渲染中。输入一个模型空间中的顶点位置，返回模型空间中从该点到光源的光照方向。没有被归一化

float3 UnityObjectToWorldDir(float3 dir) 把方向矢量从模型空间变换到世界空间中
float3 UnityWorldToObjectDir(float3 dir) 把方向矢量从世界空间变换到模型空间中
