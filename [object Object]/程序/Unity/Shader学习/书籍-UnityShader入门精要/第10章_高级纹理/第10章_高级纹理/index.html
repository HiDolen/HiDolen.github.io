之前学到的法线纹理、渐变纹理和这招纹理，都属于低维（一维或二维）纹理

立方体纹理（Cubemap）

在图形学中，立方体纹理是环境映射（Environment Mapping）的一种实现方式，可以模拟物体周围的环境。

立方体纹理最常见于天空盒子，以及环境映射。

天空盒子（Skybox）

要创建 Skybox 材质，先新建材质，然后 Shader 选择 Unity 自带的 Skybox/6 Sided。

要设置 Unity 全局的 Skybox，在 Window → Lighting → Skybox 中设置。

要单独设置摄像机显示的 Skybox，可以给摄像机添加组件 Skybox 覆盖全局设置。

环境映射

通过环境映射，可以模拟出金属质感的材质。

创建环境映射的立方体纹理，有三种方法：

反射效果（reflection）

主要使用了 o.worldRefl = reflect(-o.worldViewDir, o.worldNormal) 计算反射方向；
然后使用 fixed3 reflection = texCUBE(_Cubemap, i.worldRefl).rgb * _ReflectColor.rgb 对 Cubemap 进行采样。

详见文件 Chapter10-Reflection.shader。

折射效果（refraction）

使用 o.worldRefr = refract(-normalize(o.worldViewDir), normalize(o.worldNormal), _RefractRatio) 获得折射方向。
三个参数分别是：入射光线方向（必须归一化）、表面法线（必须归一化）、入射光线所在介质的折射率和折射光线所在介质的折射率之间的比值（η1sinθ1=η2sinθ2η_1\mathrm{sin}θ_1=η_2\mathrm{sin}θ_2η1​sinθ1​=η2​sinθ2​，角度是与法线方向的夹角）。

菲涅尔反射（Fresnel reflection）

根据视角方向控制反射程度。许多车漆、水面等材质的渲染常用菲涅尔反射来模拟。

Schlick 菲涅尔近似等式：

FSchlick(v,n)=F0+(1−F0)(1−v⋅n)5F_{Schlick}(\pmb v,\pmb n)=F_0+(1-F_0)(1-\pmb v·\pmb n)^5 FSchlick​(vv,nn)=F0​+(1−F0​)(1−vv⋅nn)5

其中，F0F_0F0​ 是反射系数，控制菲涅尔反射的强度；v\pmb vvv 是视角方向，n\pmb nnn 是法线方向。

Empricial 菲涅尔近似等式：

FEmpricial(v,n)=max(0,min(1,bias+scale×(1−v⋅n)power))F_{Empricial}(\pmb v,\pmb n)=max(0,min(1,bias+scale×(1-\pmb v·\pmb n)^{power})) FEmpricial​(vv,nn)=max(0,min(1,bias+scale×(1−vv⋅nn)power))

其中，biasbiasbias scalescalescale powerpowerpower 是控制项。

渲染纹理（Render Texture）

可以把摄像机的渲染结果先放到一个中间缓冲，而不是直接显示到屏幕上。这个中间缓冲叫做渲染目标纹理（Render Target Texture, RTT），传统的是帧缓冲（后备缓冲，back buffer）。

Unity 为渲染目标纹理 RTT 定义了专门的纹理类型：渲染纹理（Render Texture）。

使用渲染纹理有两种方法：

  • 在 Project 视图下右键 → Create → Render Texture，将这个文件拖到摄像机的 Camera 组件的 Target Texture 属性
  • 在 Shader 代码里使用 GrabPass 抓取屏幕图像
    • 若使用 GrabPass { }，后续 Pass 用 _GrabTexture 访问屏幕图像，Unity 会消耗大量性能为每个物体都抓取一次屏幕图像
    • 若使用 GrabPass { "TextureName" }，后续 Pass 用 TextureName 访问屏幕图像，Unity 对于相同的纹理名称只会采样一次，所有这个纹理名称的纹理共用一张屏幕图像
